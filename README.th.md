# omise-android-example

##วิธีการติดตั้งและทดสอบ OmiseApp
### ติดตั้ง :
 - ดาวน์โหลด หรือ โคลน `OmiseApp` และ ทำการ Import project เข้า Android SDK tool.                                         
 - ติดตั้ง JDK 1.8++.                                                                                                

### ทดสอบ :
